LeetCode Math Explorer
LeetCode Math Explorer is a Streamlit app for browsing, filtering, and tracking math-related LeetCode problems. It loads cached question data, supports refresh from LeetCode, and lets users filter by difficulty, acceptance rate, topics, premium status, and finish status. The app includes card, table, and analytics views, plus progress metrics for solved and remaining questions. To run locally, install dependencies from requirements.txt, then execute streamlit run Questions.py. User progress is stored in data/user_state.json, while question cache lives in data/questions.json. Ideal for structured math practice and interview preparation.
